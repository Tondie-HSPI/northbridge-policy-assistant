# Evaluation Notes

This project can be evaluated with three categories of questions.

## Supported Questions

Questions where relevant source material exists in the Northbridge documents. The assistant should answer directly and cite sources.

## Unsupported Questions

Questions where no internal documentation supports an answer. The assistant should escalate instead of guessing.

## Ambiguous Questions

Questions where some documentation is relevant but not enough to provide a complete answer. The assistant should provide what is supported and identify uncertainty.

## Suggested Metrics

- Retrieval relevance.
- Citation quality.
- Escalation accuracy.
- Hallucination avoidance.
- Usefulness of answer to staff workflow.
